三、图片钝化

在页面中我的图像显得方方正正有点突兀，因此本人想“钝化”我的头像四角，使整个画面显得柔和一点。

软工视角：
所谓的“钝化”，本质上是修改 CSS 的 border-radius 属性。在 AirCloud 主题中，找到对应的 .avatar 类并添加以下代码即可：

css
.avatar {
border-radius: 50% !important; / 强制转为圆形 /
object-fit: cover; / 防止图片拉伸 /
}
