三、图像钝化

在页面中我的图像显得方方正正有点突兀，因此本人想钝化我的头像四角使得整个画面显得柔和一点

“钝化”，从软工的表现层来看，实际上是指将头像改为圆形（Rounded Corner/Circular），让边缘不再那么尖锐。这在 AirCloud 主题中是一个非常经典且重要的 Layout 调整。

作为软工学生，我们得去修改它的“视觉控制层”，也就是 CSS (或 Stylus/Less)。

🛠️ 实现圆形头像的终极方案

AirCloud 的头像通常是一个正方形的 <img> 标签。我们要利用 CSS 的 border-radius 属性，将其设置为百分比。

1. 找到正确的 CSS 文件

你需要去主题目录下寻找控制侧边栏样式的样式表。通常在：

  • themes/aircloud/source/css/ 目录下。
  • 寻找 aircloud.css (主文件)。
