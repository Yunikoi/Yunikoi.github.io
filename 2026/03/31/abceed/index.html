日语

4.2

新完全语法第一课

〜際（に）

要点： 相当于“〜の時”，但语气更正式，常用于公共场所的告示、说明书或商务场合。

接续： 名词+の / 动词辞书形、た形

〜に際して／〜にあたって

要点： 用于表示某个重大的、正式的活动或行为开始前。

〜たとたん

要点： 表示 A 发生后，几乎同时发生了出乎意料的 B。

接续： 必须接动词的 「た形」。后面不能接自己的意志、请求或命令。

〜かと思うと

要点： 表示前一个动作刚结束，紧接着发生了出乎意料的变化。

特点： 强调意外感或前后的剧烈对比。

限制： 不能用于描述说话人自己的动作。

〜か〜ないかのうちに

要点： 表示 A 刚刚完成或几乎还没完成的瞬间，B 就紧接着发生了。

特点： 强调时间的紧迫感，仿佛 A 和 B 是同时发生的。

接续： 动词辞书形 + か + 动词ない形 + かのうちに。

语法句型 中文意思 接续 核心特点 / 使用限制
～際（に） 在…… 的时候（正式） 名词 + の动词辞书形 / た形 书面、正式，多用于告示、说明书、商务
～に際して 在…… 之际 / 值此…… 之时 名词 + に際して动词辞书形 + に際して 用于重大、正式场合（开业、出发、就任等）
～たとたん 刚一…… 就…… 动词た形 强调瞬间、意外；后面不能接意志、请求、命令
～かと思うと 刚…… 就……（没想到） 动词た形 + かと思うと 强调意外、反差；不能用于自己的动作
～か～ないかのうちに 刚一…… 就……（几乎同时） 辞书形＋かない形＋かのうちに 强调时间极短、紧迫，动作几乎同时发生

男：鈴木さん、この間頼まれたレポートのチェックだけど。

大学で女の学生と男の学生が話しています。女の学生はこのあと何をしますか。

女：先輩、去年の夏休みに農業体験行かれたんですよね。大学が主催してる。あれ、私も来週から一か月行くんです。何か持ってったほうがいいものとかありますか。

男：保険証。去年忘れた人がいて大変だったんだよ。コピーすら持ってきてなくて。僕らみたいな素人が農業をやるときは、思わぬ怪我とかあるから忘れないように気をつけて。

女：はい、いつも持ち歩いてます。

男：そう。それから、T シャツは多めに持ってったほうがいいよ。作業着の中に着るんだけど、作業をしてると想像以上に汗を掻くからね。作業着とか長靴は貸してもらえるよ。よく実習生が来るからか、いろんなサイズのがあったよ。

女：よかった。長靴買わなきゃって思ってたけど。T シャツは、作業着があるからいいかなって思って、部屋着ぐらいしか用意してませんでした。

男：そうそう。実は去年、僕、健康診断を受けるのを忘れてて、もう少しで参加できなくなりそうだったんだよ。

女：へえ、で、どうしたんですか。

男：診断書をすぐ出してくれる病院を探して、なんとか間に合ったんだけどね。

女：ああ、それで今年は提出が先週までだったんですね。

新完全语法第二课

1.～最中だ / ～最中に

  • 中文意思：正在……当中；在……过程中
  • 接续：名词+の / 动词ている形 + 最中だ
  • 核心特点/使用限制：
    1. 表示动作正在进行的状态；
    2. 多用于短时间动作，后句常接「意外发生的事」打断当前动作；
    3. 例句：田中さんは今考えごとをしている最中だから、じゃましないほうがいい。（田中现在正在思考，最好不要打扰他）

2. ～うちに（A用法：趁着……）

  • 中文意思：趁着……（还没……的时候）
  • 接续：名词+の / 动词辞书形/ている/ない形 / イ形容词い形 / ナ形容词な形 + うちに
  • 核心特点/使用限制：
    1. 强调「趁时间/状态还没变化，赶紧做某事」；
    2. 后句必须接意志性动作（自己主动做的事）；
    3. 例句：忘れないうちに、カレンダーにメモしておこう。（趁还没忘，先记在日历上吧）

3. ～うちに（B用法：在……期间/不知不觉……）

  • 中文意思：在……期间；不知不觉……
  • 接续：动词辞书形/ている/ない形 + うちに
  • 核心特点/使用限制：
    1. 表示在某段时间内，发生了自然的变化；
    2. 后句接非意志性变化（不受人控制的事）；
    3. 例句：知らないうちに、雨が降り始めていた。（不知不觉间，下起雨来了）

4. ～ばかりだ / ～一方だ

  • 中文意思：越来越……；一味地……
  • 接续：动词辞书形 + ばかりだ / 一方だ
  • 核心特点/使用限制：
    1. 表示某一方向的变化持续发展；
    2. 「ばかりだ」多用于负面趋势（越来越糟），「一方だ」为中性；
    3. 例句：このごろは仕事が多くて残業が増えるばかりだ。（最近工作多，加班越来越多）

5. ～（よ）うとしている

  • 中文意思：即将……；正要……
  • 接续：动词う・よう形 + としている
  • 核心特点/使用限制：
    1. 表示「某事马上就要发生」的瞬间状态，语气偏硬；
    2. 只能接瞬间动词（开始、开幕、完成等）；
    3. 例句：さあ、決勝戦が今、始まろうとしています。（好了，决赛现在即将开始）

6. ～つつある

  • 中文意思：正在……；逐步……
  • 接续：动词ます形 + つつある
  • 核心特点/使用限制：
    1. 表示变化正在持续进行中，书面、正式语气；
    2. 多用于渐进式变化（变暖、发展、近代化等）；
    3. 例句：次第に暖かくなりつつあります。春はもうすぐです。（天气正逐渐变暖，春天马上就到了）

7. ～つつ

  • 中文意思：一边……一边……；同时……
  • 接续：动词ます形 + つつ
  • 核心特点/使用限制：
    1. 表示两个动作同时进行，书面、正式语气；
    2. 前后句必须是同一主语；
    3. 例句：いろいろな体験を楽しみつつ、日本の生活に慣れていった。（一边享受各种体验，一边习惯了日本的生活）
