  • ～料（服务费）：通行料、入場料、授業料、手数料。
  • ～代（代替物/消耗）：ガス代、食事代、部屋代、修理代。
  • ～費（特定的用途经费）：生活費、参加費、事務費、宣伝費。
  • ～賃（劳动力或租赁）：家賃、手間賃、電車賃。

2. 性质/状态类

  • ～感（感觉）：安心感、違和感、満足感、責任感。
  • ～性（性质）：可能性、安全性、危険性、効率性。
  • ～化（变成某状态）：国際化、映画化、機械化、少子化。
  • ～的（……的）：健康的、積極的な、比較的、具体的。

3. 数据/统计类

  • ～率（比例）：進学率、合格率、失業率、普及率。
  • ～量（多少）：降水量、生産量、消費量、仕事量。
  • ～数（个数）：利用者数、人数、経験数。
  • ～度（程度）：認知度、信頼度、満足度、自由度。

4. 团体/机构/制度类

  • ～制（制度）：予約制、会員制、担任制。
  • ～界（领域）：芸能界、医学界、スポーツ界。
  • ～圏（范围）：首都圏、北極圏、文化圏。

5. 其他高频后缀

  • ～別（按……分）：学年別、男女別、地域別。
  • ～風（风格）：北欧風、和風、サラリーマン風。
  • ～中（范围内/动作中）：世界中、体中、営業中。

  • 不～（不……）：不可能、不採用、不自然、不器用。
  • 無～（没有）：無意識、無責任、無関心、無計画。

  • 準～（准/次级）：準決勝、準急、準会員。
  • 新～（新）：新発売、新体制、新年度。

  1. 范围/程度类

import numpy as np    # 引入 NumPy：负责生成和计算数字
import pandas as pd   # 引入 Pandas：负责把数字做成表格（Excel）

# --- 第一步：定义“生成走路数据”的模版 ---
def generate_walking_data(duration_seconds=100):
    """
    模拟正常走路。手机在兜里会有轻微的、有节奏的晃动。
    """
    # 每一秒钟手机产生 10 个数据点（频率 10Hz）
    samples = duration_seconds * 10
    
    # 模拟 x, y, z 三个轴的震动
    # np.random.normal(中心值, 摆动幅度, 数量)
    # 这里的摆动幅度只有 0.5，说明波动很小
    x = np.random.normal(0, 0.5, samples)
    y = np.random.normal(9.8, 0.5, samples) # Y轴通常由于重力接近 9.8
    z = np.random.normal(0, 0.5, samples)
    
    # 把这些数字装进一个叫 DataFrame 的表格里
    df = pd.DataFrame({'x': x, 'y': y, 'z': z})
    
    # 贴标签：0 代表“正常”
    df['label'] = 0
    return df

# --- 第二步：定义“生成摔倒数据”的模版 ---
def generate_fall_data(num_falls=10):
    """
    模拟剧烈摔倒。加速度会瞬间产生一个巨大的“峰值”。
    """
    fall_list = []
    for _ in range(num_falls):
        # 摔倒瞬间，数值会突然从 0.5 暴增到 15 甚至更高
        impact_x = np.random.normal(0, 15, 5) # 持续 0.5 秒的冲击
        impact_y = np.random.normal(0, 15, 5)
        impact_z = np.random.normal(0, 15, 5)
        
        df = pd.DataFrame({'x': impact_x, 'y': impact_y, 'z': impact_z})
        
        # 贴标签：1 代表“摔倒”
        df['label'] = 1
        fall_list.append(df)
        
    # 把所有的摔倒片段粘在一起
    return pd.concat(fall_list)

# --- 第三步：开始正式“生产”数据 ---

# 1. 生产 1000 秒的走路数据
walking_data = generate_walking_data(1000)

# 2. 生产 50 次摔倒的数据
fall_data = generate_fall_data(50)

# 3. 把走路和摔倒的数据拼成一张超级大表
final_dataset = pd.concat([walking_data, fall_data])

# 4. 把这张大表存成电脑里的一个 CSV 文件（你可以用 Excel 打开它）
# index=False 意思是不要在左边生成多余的行号
final_dataset.to_csv('sentio_raw_data.csv', index=False)

raw_data['mag'] = np.sqrt(raw_data['x']**2 + raw_data['y']**2 + raw_data['z']**2)

# --- 3. 准备两个新筐子：X 装题目，y 装答案 ---
features = []  # 题目：这段时间的平均力量和波动程度
labels = []    # 答案：这段时间到底是 0（正常）还是 1（摔倒）

# --- 4. 滑动窗口：像放大镜一样扫描数据 ---
# window_size = 10 意味着我们每次看 10 行（大约 1 秒的数据）
window_size = 10

for i in range(0, len(raw_data) - window_size, 5): # 每次挪动 5 行，增加重叠度
    # 截取这 10 行数据
    window = raw_data.iloc[i : i + window_size]
    
    # 提取特征：把 10 个数字浓缩成 2 个核心指标
    mean_mag = window['mag'].mean()  # 这 1 秒的平均力量
    std_mag = window['mag'].std()    # 这 1 秒的波动剧烈程度（标准差）
    
    # 获取这段时间的“标准答案”
    # 如果这 10 行里有任何一行标记为 1，我们就认为这一秒发生了摔倒
    label = 1 if window['label'].any() == 1 else 0
    
    # 装进筐里
    features.append([mean_mag, std_mag])
    labels.append(label)

# --- 5. 转化为 AI 专用的格式并保存 ---
X = np.array(features)
y = np.array(labels)
